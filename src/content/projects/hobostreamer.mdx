A fully live, community-first streaming platform at [hobostreamer.com](https://hobostreamer.com). Start instantly in the browser or connect OBS/ffmpeg, then restream simultaneously to Twitch, YouTube, Kick, and any RTMP server from a single dashboard.

Features: real-time chat with custom emotes, automatic clip detection, auto-recorded VODs, built-in HoboQuest game for viewers, sitewide global chat, voice channels, pastes & file sharing with syntax highlighting, full stream analytics, and comprehensive moderation tools. Every line of code is public on GitHub.
