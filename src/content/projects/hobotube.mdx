A minimal-distraction desktop YouTube client built around [mpv](https://mpv.io) for buttery 4K playback. Features a proper persistent queue, collection management, hardware acceleration detection, auto-updater via GitHub Releases, and an in-app diagnostics panel. Built as a personal tool to complement the HoboStreamer clip pipeline.
